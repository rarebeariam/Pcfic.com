# Pcfic.com

A professional, responsive website with modern design.

## Features

- Responsive design that works on desktop, tablet, and mobile
- Modern gradient hero section
- Navigation menu with smooth scrolling
- About section
- Services showcase with cards
- Contact form
- Professional styling with CSS

## How to Run

### Option 1: Python HTTP Server (Recommended)
```bash
python3 -m http.server 8000
```
Then open your browser and visit: `http://localhost:8000`

### Option 2: Using any other HTTP server
You can use any HTTP server of your choice. For example:
- Node.js: `npx http-server`
- PHP: `php -S localhost:8000`

### Option 3: Direct File Opening
Simply open `index.html` in your web browser (Note: some features may not work with file:// protocol)

## Files

- `index.html` - Main HTML file with website structure
- `styles.css` - CSS stylesheet with all styling

## Customization

Edit the HTML content in `index.html` and styles in `styles.css` to customize the website for your needs.
